# TV DY

抖音 Android TV 适配版 — 通过 WebView 加载抖音网页版，为遥控器操作优化的 TV 端体验。

## 功能特性

- 🎮 **遥控器适配**：完全适配 D-pad 方向键遥控器操作
- 🔘 **功能轮盘**：左右键呼出圆形功能轮盘，支持点赞、收藏、关注、评论等
- 💬 **弹幕支持**：支持弹幕开关控制
- 🖥️ **全屏模式**：沉浸式全屏体验，隐藏状态栏和导航栏
- ⚡ **快捷操作**：
  - 上/下键：切换视频
  - OK/确认键：播放/暂停
  - 左/右键：呼出功能轮盘
  - 菜单键：显示遥控指南
  - 返回键：双击退出

## 功能轮盘操作

| 功能 | 说明 |
|------|------|
| 点赞 | Z 键 — 赞 / 取消赞 |
| 收藏 | C 键 — 收藏 / 取消收藏 |
| 关注 | G 键 — 关注 / 取消关注 |
| 评论 | X 键 — 打开评论 |
| 弹幕 | B 键 — 弹幕开关 |
| 清屏 | J 键 — 清屏模式 |
| 全屏 | Y 键 — 网页内全屏 |

## 技术实现

- **WebView** 加载 `douyin.com` 网页版，设置桌面端 User-Agent
- 通过 JavaScript 注入模拟键盘事件，实现精准的快捷键控制
- 自定义圆形轮盘 UI，带自动隐藏和动画效果
- 最低支持 Android 4.4 (API 19)，目标 Android 15 (API 35)

## 构建

使用 Android Studio 或 Gradle 命令行：

```bash
./gradlew assembleRelease
```

APK 输出路径：`app/release/app-release.apk`
