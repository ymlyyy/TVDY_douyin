# TV DY

抖音 TV 版，遥控器刷抖音。

## 按键说明

| 按键 | 功能 |
|------|------|
| 上/下 | 切换视频 |
| OK | 播放 / 暂停 |
| 左/右 | 呼出功能轮盘 |
| 菜单 | 显示按键指南 |
| 返回 | 双击退出 |
